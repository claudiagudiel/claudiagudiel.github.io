Selección por etiqueta CG

Con la selección por etiquetas podemos seleccionar los encabezados a los que queremos dar un color Ejemplo: h2 {color:();}. Las etiquetas h2 están con color.

Selección por id CG

Para dar color a una etiqueta. Ejemplo:#id {background-color:();} Este texto está con color de fondo usando id.

Selección por class CG

Para dar color a una etiqueta del HTML. La diferencia con id es que a este le precede un . y al id un #. Texto con color usando class, dando como nombre "class"
