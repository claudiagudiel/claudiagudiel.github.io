Fuente genérica

La propiedad font-family puede tomar como valor el nombre concreto de una fuente. El nombre de la fuente debe escribirse entre comillas si contiene espacios (y puede escribirse entre comillas si no contiene espacios). Si el ordenador no tiene instalada la fuente indicada, el navegador utilizará una fuente de la familia genérica serif, como sería en el siguiente caso:

Texto escrito en sans serif

Fuente específica

En este caso queremos que el texto tenga una letra especifica y no una genérica, esta sera visible siempre y cuando el usuario tenga dicha fuente descargada en su dispositivo.

Tamaño de fuente en px

Podemos variar el tamaño de la fuente especificando el tamaño en píxeles que queremos darle.

Tamaño de fuente en em

También podemos variar el tamaño del texto con em, en este caso se multiplica o se divide el texto por el número especificado.

La propiedad font-style permite establecer el estilo con el que se muestra el tipo de letra seleccionado mediante la propiedad font-family.
